## CSS 传统定位与布局

- display: block; 块
- display: inline; 行内
- display: inline-block; 行内块
- position: static; 静态定位
- position: relative; 相对定位
- position: absolute; “绝对”定位
- position: fixed; 固定定位
- float 浮动
- clear 清除浮动
